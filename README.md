# GPA Genie

A PHP-based web application to calculate and manage GPA, including a blog section.

## Features
- GPA Calculator
- Blog posts connected to a MySQL database
- Login/Signup system

## Setup
1. Import `database/gpagenie.sql` into your MySQL DB
2. Update `config.php` with your DB credentials
3. Host it using services like 000WebHost or InfinityFree
